I'm Mike

My name is Mike Schmidt. My journey to becoming a web developer started in 2020 when the pandemic hit. While working as a law enforcement officer, I realized I wanted to make a career change so my wife suggested that we should start learning how to code. I started listening to podcasts and was inspired hearing about people who were able to make positive impacts for others just by writing code. I reached out to a friend who is a front end developer and he suggested I start by reading Murach's "HTML 5 and CSS 3" to get a basic understanding of building an HTML website. I read through that and later read Jon Duckett’s "HTML & CSS" and "JavaScript & JQuery." Once I became more familiar with JavaScript I purchased O'Reilley's "JavaScript The Definitive Guide 7th Edition" for supplemental reading on the subject.

I then delved into courses on Udemy. I started with Rob Percival’s “The Complete Web Developer Course 2.0” to learn basic HTML, CSS, and Javascript. This course was a bit outdated so I then moved to Dr. Angela Yu’s “The Complete 2021 Web Development Bootcamp” and followed the lessons through the front-end developer course. I loved everything I was learning about web development and I wanted to dive deeper into everything JavaScript was capable of, so I purchased Jonas Schmedtmann’s “The Complete JavaScript Course 2021: From Zero to Expert!” I am currently learning React using the tutorials from Wes Bos on React for Beginners and Advanced React.

I spend countless hours during my nights and weekends learning web development and I love doing it! I enjoy improving my code and making my websites even faster and more responsive.
